-- Predictive Maintenance (Tahminleyici Bakım) Projesi--

Bu proje, endüstriyel makinelerin sensör verilerini kullanarak arızaları önceden tahmin etmek ve bakım süreçlerini optimize etmek amacıyla geliştirilmiştir. Proje kapsamında hem Python hem de KNIME üzerinde veri bilimi çalışmaları yapılmıştır.

--Uygulanan Analizler ve Modeller--
* Sınıflandırma (Classification): Makinelerin arıza yapıp yapmayacağının tahmini.
* Regresyon (Regression): Kalan Faydalı Ömür (RUL) süresinin hesaplanması.
* Kümeleme (Clustering): Benzer çalışma karakteristiklerine sahip makinelerin gruplandırılması.
* Anomali Tespiti (Anomaly Detection): Normal dışı sensör hareketlerinin belirlenmesi.
* Birliktelik Kuralları: Hata türleri arasındaki ilişkilerin analizi.

-- Kullanılan Teknolojiler--
* Diller: Python (Pandas, Scikit-learn, Matplotlib, Seaborn)
* Araçlar: KNIME, Jupyter Notebook, GitHub
* Algoritmalar: Random Forest, Linear/Logistic Regression, K-Means

-- 📂 Dosya Yapısı--
* Predictive_Maintenance.ipynb: Python analiz kodları.
* KNIME_Workflows: Projenin KNIME üzerindeki iş akışları.
* datasets: Kullanılan veri setleri.


-- Predictive Maintenance Project--

This repository contains a comprehensive Predictive Maintenance study developed to predict equipment failures and optimize maintenance schedules using sensor data. The project explores end-to-end data science workflows using both Python and KNIME.

-- Key Features & Models--
* Classification: Predicting failure states and equipment status.
* Regression: Estimating **Remaining Useful Life (RUL)** of machinery.
* Clustering: Grouping machines with similar operational characteristics.
* Anomaly Detection: Identifying irregular sensor patterns and outliers.
* Association Rule Mining: Analyzing relationships between different failure types.

--Tech Stack--
* Languages: Python (Pandas, Scikit-learn, Matplotlib, Seaborn)
* Tools: KNIME, Jupyter Notebook, Git/GitHub
* Algorithms: Random Forest, Linear & Logistic Regression, K-Means Clustering

--  📂 Project Structure--
* Predictive_Maintenance.ipynb: Core analysis and model training in Python.
* KNIME_Workflows: Visual data workflows exported from KNIME.
* data: Datasets used for training and testing.
